AyfaCare – Healthcare Appointment System

Project Overview
AyfaCare is a web-based healthcare appointment system developed using ASP.NET Web Forms. The system connects patients with healthcare providers, manages appointment bookings, and handles both public and private sector healthcare services.

Features

Patient Features
- Client booking and registration
- Appointment confirmation
- Profile management
- View booked appointments

Doctor Features
- Doctor registration
- Profile management (DoctorEditPro.aspx)
- Time slot management
- View booked clients

Admin Features
- Admin dashboard
- Manage doctors and patients
- System oversight

Key Pages
Default.aspx – Home page
ClientBooking.aspx – Patient appointment booking
AppointConfirm.aspx – Appointment confirmation
BookedClients.aspx – View booked appointments
DocReg.aspx – Doctor registration
DoctorEditPro.aspx – Edit doctor profile
AdminLog.aspx – Admin login
AdminDash.aspx – Admin dashboard
